# docker-phantomjs-webdriver-standalone

Docker container to run phantomjs webdriver standalone.


![](https://codeclou.github.io/doc/docker-warranty.svg?v5)

-----

### Usage

```
docker run \
    -d \
    -p 4444:4444 \
    codeclou/docker-phantomjs-webdriver-standalone:latest
```
